# Libras Connect 

## 1. Sobre o Projeto

* Aplicativo para conectar pessoas surdas e ouvintes que sabem Libras.
* Frontend: Expo + React Native.
* Backend: Node.js + Express.
* Testável diretamente no aplicativo Expo Go.

## 2. Idioma

* Todo o aplicativo deve estar em português do Brasil.

## 3. Tecnologias Utilizadas

* Frontend: Expo + React Native
* Backend: Node.js + Express
* Chat em tempo real: Socket.io
* Armazenamento de mídia: Cloudinary
* Vídeo-chamada: LiveKit
* Inteligência Artificial: OpenAI (Whisper e GPT-4o)
* Envio de e-mail: Resend
* Pagamentos: Mercado Pago (Pix)
* Monitoramento externo: UptimeRobot
* Banco de dados: Firebase
* Ambiente de desenvolvimento/hospedagem: Replit
* Controle de versão: GitHub

## 4. Monetização

* 100% via Pix, sem cartão de crédito.

## 6. Regras Gerais do Projeto

* Não inventar código, APIs, chaves ou configurações.
* Nunca substituir uma API por outra.
* Nunca trocar uma biblioteca sem autorização.
* Nunca remover funcionalidades sem autorização.
* Se algum recurso não puder ser concluído, explicar claramente o motivo e continuar desenvolvendo as demais partes do projeto.
* Tudo deve funcionar de verdade — nenhum botão pode ficar apenas como desenho.
* Todas as funções devem executar ações reais.
* Se uma função depender de API externa, parar e perguntar antes de continuar.
* Usar boas práticas de programação, organizar os arquivos, escrever código limpo e reutilizável.
* Ao final do desenvolvimento, explicar quais chaves ou configurações ainda precisam ser fornecidas.
